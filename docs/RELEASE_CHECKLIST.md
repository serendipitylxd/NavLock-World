# Public Release Checklist

Before making this repository public:

- [ ] Confirm that no dataset files are included.
- [ ] Confirm that no model weights are included.
- [ ] Confirm that no `outputs/`, caches, logs, or local run files are included.
- [ ] Confirm that no private `doc/`, `latex/`, or raw `fig/` working files are included.
- [ ] Decide and add a formal open-source license.
- [ ] Update citation metadata after the paper author list and venue status are final.
- [ ] Run the tests that do not require private data or model weights.
